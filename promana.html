計算問題

PV (Planned Value) : 予想利益
EV (Earned Value) : 利益
SV (Schedule Variance) : スケジュール差異
SPI (Schedule Perfomance Index) : スケジュール効率指数

PV: 500人 / 月
EV: 300人 / 月 とすると、
SV = PV - EV = -200
SPI = EV / PV = 0.6

AC (Actual Cost) : かかったコスト
CV (Cost Variance): コスト差異
CPI (Cost Performance Index): コスト効率指数

AC: 400人 / 月
EV: 200人 / 月 とすると、
CV = EV - AC = -200
CPI = EV / AC = 0.5
